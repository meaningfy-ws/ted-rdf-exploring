Jupyter Notebook - R

This section shows an example using Jupyter Notebook in R language. R is software environment for statistical computing and graphic representation. In this case, R represents the programming language for writing notebook documents. To realize the proposed scenario, it is necessary to install the special tools (Running R Jupyter Notebooks) and use the R code that will perform a query (SPARQL with R) to the cellar and display the results in tabular form.

To run R sample:

  1. Install & install R language:

    • Windows 64bit: download

  1. Download Jupyter Notebook file: https://github.com/OP-TED/ted-rdf-conversion-pipeline/blob/main/notebooks/query_cellar_R.ipynb

  1. Download SPARQL package archive and put in same folder as Notebook file: download SPARQL

  1. Install IRKernel:

  1. Run R in OS command line and type:

    install.packages("IRkernel")
install.packages('RCurl')
install.packages('XML')

  1. In appeared window, select the first mirror and press OK:

  1. After installation is completed, type:

    IRkernel::installspec()

  1. Open Jupyter Notebook file in your favourite code editor (e.g VS Code);
